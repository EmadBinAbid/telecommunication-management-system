Modules Description:
____________________

* Module 1:
	Opening screen of the system. It takes user's login credentials and allows him/her the system's access. Additionally, it contains a "Quit" button to exit the application instantly. 

* Module 2:
	This module allows users to access the information stored based on their choice. Specifying any choice will navigate the user to appropriate screens. 

* Module 3:
	Displays the information about staff members.

* Module 4:
	Displays information about customers.  

* Module 5:
	Allows new staff to enter into database.

* Module 6:
	Allows new customer to enter into database.
